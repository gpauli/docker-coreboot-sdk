Coreboot sdk image
===

SDK for developing coreboot. Based on the original with additions

# Requirements

docker-debian-mini Image see https://github.com/gpauli/docker-debian-mini.git

# Building

* change variables in build.sh
  * mybase
  * mycoreboot
  * mytag
  
* run build.sh
